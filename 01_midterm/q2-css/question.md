### Question 2: 

What do the following CSS selectors select? Please provide short examples

    div + p
    div p
    div, p
    div > p
    div ~ p


### Your Answer:

div + p - Selects all the <p> elements that are immediatly after <div> elements
div p - Selects all the <p> elements that are inside of <div> elements
div, p - Selects all <div> elements & all <p> elements
div > p - Selects all <p> elements that have <div> as a parent element
div ~ p - Selects every <p> element that is preceded by a <div> element
